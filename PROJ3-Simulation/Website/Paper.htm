There. We went from two blocks and a wall to the digits of pi, and all we had to do is count the number of collisions. This gets better and better the heavier m2 gets, as shown in this graph.



However, this method of calculation very quickly becomes impossible to replicate. First off, you need a system with no friction and perfectly elastic collisions, which is already basically impossible. Additionally, to calculate just 12 digits of pi, you would need an object with mass 1024 kg, which is equal to the mass of the Earth. You would also need a way to count the collisions without introducing any friction. The point is, this is a very impractical, but interesting, way to calculate pi.
